Units

em vs. rem

Use rem for text sizes to avoid cascading problem, i.e. all text sizes will be relative to html tag's size.

Use em for margin and padding.

Use em or percentage for width.

em for margins, etc. are relative to size of "this" element - not parent element! This is useful when using em for margins, etc. as they also scale.

Pixels

Pixels are now defined by reference pixel, 1/96 inch, to avoid problems with large resolution screens.

justify-content vs. align-items

Use justify-content to spread content to whole width of container ("row") by adding space between or around elements ("columns").

Use align-items to specify how to spread content vertically - default is stretching to same height as parent. A useful value is align-items: start to not spread to full height.
